## 🛠️ Tech Stack & Architecture
This portfolio is built using modern frontend design principles without heavy frameworks, achieving near-instant load times and premium aesthetics:
- **Core Structure**: HTML5 (semantic layout & SEO optimized)
- **Styling**: Vanilla CSS3 (custom CSS design system, glassmorphism, responsive flex/grid layouts)
- **Interactivity**: Vanilla JavaScript (ES6+)
- **Canvas Animations**: HTML5 Canvas particle/constellation background API
- **Form Delivery**: Web3Forms API integrations (native form routing directly to inbox)

---

## ✨ Features
1. **Interactive Portfolio Bot (Left-Side Popup)**: A custom-built, responsive assistant designed to answer visitor questions regarding my Django backend skills, Zoho Deluge workflows, AI modeling, and contact coordinates.
2. **HTML5 Constellation Particles**: An dynamic, math-based particle network background rendering real-time line connections between drifting nodes.
3. **Animated Brand Logo**: A custom circular tech emblem featuring smooth 360-degree rotation and dual-color glow transitions on cursor hover.
4. **Project Tab Filters**: Instantly filter projects by category (AI & Data Science, Django & Web, Zoho & Automation) without page reloads.
5. **Secure Contact Form**: Bypasses firewalls using Web3Forms endpoint to send client emails directly to `nandkumarcoder@gmail.com`.

## 💻 Local Setup
To run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/nandkumarcoder/nand-kumar-portfolio.git
   ```
2. Navigate into the project folder:
   ```bash
   cd nand-kumar-portfolio
   ```
3. Start a local server (e.g. Python server):
   ```bash
   python -m http.server 3000
   ```
4. Open your browser and navigate to `http://localhost:3000`.
